# passwordrecidive
Генератор паролей, который не будет тебя предавать.
И шифратор, чтобы твои секреты остались только твоими.

Что это за зверь?
Это десктопное приложение на Qt, которое делает две полезные вещи:

Генерит рандомные пароли — от «12345» до «хуйпоймешь».

Шифрует любой текст (пароль, сообщение, список покупок) по ключу. Никто не прочитает, если не знает код.

Сделано для тех, кто устал хранить пароли в заметках телефона и надеяться на «авось не взломают».

C++ 17 — без соплей, только дело.

Qt 6 — интерфейс, который не бесит.

XOR-шифрование — простой, но жесткий алгоритм. Один ключ — один текст. Дважды применил — получил обратно.

Base64 — чтобы результат шифрования был читаемым, а не набором иероглифов.

Генерация
Жмешь кнопку — получаешь пароль длиной 16 символов. Буквы, цифры, спецсимволы — всё, что нужно для выноса мозга хакерам.

Шифратор

Открывается отдельным окном по кнопке "Шифратор"

Вставляешь текст и ключ

Жмешь кнопку — получаешь зашифрованную хуйню

Повторяешь с тем же ключом — расшифровываешь обратно

Самодостаточность: программа работает без интернета и внешних библиотек.

Понятный код: структура разбита на логические части, читается даже после трех энергосов.

Честное шифрование: никаких "закрытых ключей" на сторонних серверах. Твой ключ — твоя безопасность.
